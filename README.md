# 🚀 create-mern-stack

A powerful CLI to quickly scaffold a full-stack MERN application with a production-ready backend structure and basic authentication setup.

---

## ✨ Features

- ⚡ One-command MERN setup
- ⚛️ Next.js frontend
- 🛠️ Express + MongoDB backend
- 🔐 JWT Authentication (Register/Login)
- 📁 Clean and scalable backend architecture
- 🌱 Environment variables pre-configured
- 🔌 Ready-to-use API structure

---

## 📦 Installation

### Using npx (recommended)

```bash
npx create-mern-stack my-app
